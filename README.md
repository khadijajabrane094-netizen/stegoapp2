## 📖 **عن المشروع**

**StegoApp Premium** هو تطبيق متخصص في **إخفاء الرسائل داخل الصور** باستخدام تقنية **LSB (Least Significant Bit)**.

### 🔒 **المميزات:**
- ✅ إخفاء رسائل نصية داخل الصور
- ✅ استخراج الرسائل المخفية
- ✅ واجهة مستخدم جذابة وسهلة
- ✅ يعمل على جميع الأجهزة (PC - هاتف)
- ✅ تصميم عصري وجذاب
- ✅ خلفية متحركة وتأثيرات بصرية

---

## 🚀 **التقنيات المستخدمة**

| التقنية | الوصف |
|----------|-------|
| 🐍 Python | لغة البرمجة الأساسية |
| 🎨 Streamlit | إطار العمل لبناء الواجهة |

| 🎯 LSB | خوارزمية إخفاء الرسائل |
